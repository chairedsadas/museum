# museum
python-web用Django搭建博物馆
# 提示
### 用python3.8或者3.9的环境
#### 终端安装：
##### python -m pip install --upgrade pip
##### pip install django==2.2.4
##### pip install Pillow
##### pip install pyquery
##### pip install whoosh django-haystack jieba
##### pip install django-widget-tweaks
##### pip install django-simpleui
##### pip install docxtpl
